# Story boss portraits (Dungeon+)

Файлы: `{slug}.webp`, где `slug` совпадает с колонкой `story_boss_definitions.slug` (например `act1_p5.webp`).

Путь в БД по умолчанию: `/static/game/bosses/webp/{slug}.webp`.

Положите арт с прозрачностью или на тёмном фоне; рекомендуемая высота для UI ~256–512px.
