# Шаблоны способностей монстров (соло-данж)

Таблица `monster_ability_templates` + поле `monster_templates.monster_ability_template_id`.

Состояние боя: `dungeon_runs.active_waifu_debuffs` / JSON-массив экземпляров дебаффов на основную вайфу.


| slug             | Эффект (effect_json.kind) | Смысл                                                                                                        |
| ---------------- | ------------------------- | ------------------------------------------------------------------------------------------------------------ |
| `viper_poison`   | `dot_poison`              | После первого попадания по монстру: следующие N ваших сообщений наносят фиксированный урон по ОВ каждый ход. |
| `static_shock`   | `shock`                   | После первого попадания: на следующие M сообщений шанс, что урон **по монстру** не пройдёт.                  |
| `spore_weakness` | `weakness`                | После первого попадания: следующие K успешных атак с пониженным уроном по монстру.                           |


Триггер в коде: `first_player_hit` — дебафф выдаётся после первого сообщения, которое нанесло урон по текущему монстру (не уклонение, не шок).

Разведение статов «толстый / стеклянный» — поле `dungeon_run_monsters.stat_profile`: `tank` | `balanced` | `glass`; бюджет силы см. `waifu_bot.game.monster_power`.
