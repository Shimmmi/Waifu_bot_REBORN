# Балансный анализ пассивных навыков

Методика: базовый урон **100**, входящий урон **100**. Для каждого узла — эффект на **max level** с учётом реальной логики в `combat.py` и `passive_skills.py`. Условные эффекты (шансы) оцениваются как **ожидаемое значение (EV)**.

Легенда вердиктов: **OP** — сильнее целевого диапазона; **норма** — в диапазоне; **слабый** — ниже целевого.

Целевые ориентиры:
- Плоский % урона (always-on): max **+20–25%** DPS на узел T1, **+35–50%** на T2–T3, **+60–80%** на T4.
- Шансовые защиты: EV снижения входящего **≤25%** на max (без учёта лимитов 1/данж).
- Instakill / revive / survive: EV **≤15–18%** «эквивалента HP» на max.

## Покачковый кэп по уровню ОВ

`level_cap = clamp(waifu_level - waifu_level_req + 1, 0, max_level)`

Пример T4 (waifu_level_req=35): на 35 ур. ОВ — только 1-й уровень узла; на 36 — 2-й; … на 39 — 5-й (max).
