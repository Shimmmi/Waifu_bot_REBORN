# Система зачарования v2

## Две системы

### Заточка (+1…+10) — вкладка «Заточка» в кузнице

- Оплачивается **золотом**.
- Усиливает **один канал** по типу предмета:
  - **Оружие** — урон (`enchant_dmg_step × уровень`).
  - **Броня / offhand с бронёй** — броня (`enchant_arm_step × уровень`).
  - **Кольца и амулеты** — fraction-вторичка (`enchant_sec_step × уровень`): крит, уклонение, снижение урона и т.д.
- С +8 есть риск неудачи; камень защиты смягчает откат.
- **Пробуждение:** при первой успешной заточке **+1** на аксессуаре (или passive-легендарке) без fraction-вторички выдаётся **случайная** fraction-вторичка. Passive-бонус при этом **не** растёт от заточки.

### Зачарование (craft) — вкладка «Зачарование» в кузнице

- Оплачивается **пылью зачарования** (+ немного золота в будущих версиях; сейчас только пыль).
- Работает с **fraction-каналом** на **любом** предмете:
  - **Добавить** — если fraction ещё нет.
  - **Перекатать** — новый тип и значение.
  - **Усилить** — +шаг к значению (cap по tier).
- Passive-вторичку craft **не** меняет.

## Пыль зачарования

- Ресурс игрока: `enchant_dust` (отображается в профиле и кузнице).
- Получается **распылением** предметов (кнопка «Распылить» в инвентаре/магазине).
- Распыление **не даёт золота**; продажа **не даёт пыли**.

Формула пыли (настраивается в `game_config`):

```
dust = base × rarity_mult × tier_mult^(tier-1) × (1 + enchant_level × plus_mult)
```

## Fraction-вторички

Типы: `crit_chance_pct`, `evade_pct`, `dmg_reduce_pct`, `hp_max_pct`, `exp_bonus_pct`, `gold_bonus_pct`, `magic_find_pct`.

Значения — доли (0.01 = +1%).

## Snapshot на экземпляре

Вторички хранятся на `inventory_items`:

| Поле | Назначение |
|------|------------|
| `secondary_bonus_type/value` | Passive / основная вторичка |
| `secondary_fraction_type/value` | Fraction для заточки и боя |
| `secondary_awakened` | Fraction выдана пробуждением на +1 |

При генерации предмета значения копируются из шаблона; awaken и craft меняют instance.

## Баланс (стартовые ключи)

- `enchant.awaken.base_min`, `enchant.awaken.base_per_tier`
- `dismantle.*` — распыление
- `craft.*` — стоимость и cap fraction

Точные числа — в таблице `game_config`, без деплоя кода.
