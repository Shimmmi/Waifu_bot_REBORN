## Общие ограничения (обязательны)

- Пиши **только на русском языке**.
- Это презентационный бриф для **ИИ-агента**, который будет переносить игру с Telegram WebApp на Steam.
- **ЗАПРЕЩЕНО:** числовые балансы, коэффициенты, таблицы формул, конкретные проценты и лимиты из game_config.
- **ЗАПРЕЩЕНО:** схема БД, имена таблиц/колонок, SQL, ORM-модели, ID сущностей из seed.
- **ЗАПРЕЩЕНО:** секреты, токены, .env.
- **ЗАПРЕЩЕНО:** полный перечень API-роутов — только концептуальные группы.
- **ЗАПРЕЩЕНО:** упоминать legacy GD (`/gd_start`, `/engage`) — только **GD v1**.
- Не пиши «Эксперт 1 сказал», «Judge», артефакты fusion — только финальный связный текст.
- При необходимости сослаться на баланс: «детали баланса см. COMBAT_FORMULAS / game_config (не включено в этот документ)».
- Используй markdown: заголовки H2/H3, списки, таблицы где уместно, mermaid-диаграммы где указано.
- Не добавляй оглавление раздела — оно будет в финальном документе.
